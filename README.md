# AP_LabExercise_Mussie_Tadesse

This workspace includes three Java projects created for lab exercises and demonstrations.

## chatApp
A messaging application with both server and client components.  
Build files and run scripts are located inside the `chatApp/` directory.

## Notepad
A simple GUI-based notepad application.  
You can find build and execution scripts in the `Notepad/` folder.

## PokerGame
A Java-based poker game featuring AI, game engine logic, and a graphical user interface.  
The source code and tests are located in the `PokerGame/` directory.
